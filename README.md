## Smart Expense Tracker

A secure full-stack web application to manage personal income and expenses with real-time balance tracking and JWT-based authentication.

## Features
User registration and login with JWT authentication
Add, view, and delete income and expense transactions
Real-time balance calculation dashboard
Transaction history with date tracking
Protected APIs using token-based authorization
Tech Stack

Frontend: React, Tailwind CSS, Axios
Backend: Node.js, Express.js
Database: PostgreSQL
Authentication: JWT, bcrypt

## How to Run
Backend
cd backend
pnpm install
node server.js
Frontend
cd frontend
pnpm install
pnpm run dev
